# STarfield GMST editor

A simple UI to edit Starfield GMSTs.

## Features

- filter GMSTs
- revert to default
- mod integration
- merge mods
- view conflicts

## Usage

- download the zip file
- extract anywhere
- run `sf_gmst_editor.exe`
- edit the GMSTS you like
- click "Save"
